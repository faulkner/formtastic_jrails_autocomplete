h1. FormtasticAutocomplete

An extension for Formtastic[1] to generate AJAX autocomplete fields using
3rd party autocomplete plugins (unobtrusive javascript enable or not).

h2. Example

h3. Unobtrusive version (using jrails_auto_complete[2])

<pre><code><% semantic_form_for @user do |form| %>
  <% form.inputs do %>
    <%= form.input :name, :as => :autocomplete, :min_length => 3 %>
  <% end %>
<% end %></code></pre>

h3. Classic AJAX version (using ?)

<pre><code><% semantic_form_for @user do |form| %>
  <% form.inputs do %>
    <%= form.input :name, :as => :autocomplete, :url => autocomplete_task_name_path %>
  <% end %>
<% end %></code></pre>

h2. Dependencies

You must have the formtastic gem added to your application + an autocomplete
gem/plugin present and the :defaults javascript must be included in your layout.
